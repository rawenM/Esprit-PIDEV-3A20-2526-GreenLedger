🚀 POUR TESTER

1. Compiler avec votre IDE (Build Project)

2. Lancer l'application

3. Tester le scroll:
   - Si le contenu dépasse la fenêtre
   - Vous pouvez scroller avec la molette
   - Ou avec la barre de défilement

4. Tester le puzzle:
   - Cliquer "🧩 Captcha Puzzle"
   - Vous devriez voir:
     • Image de fond avec un trou
     • Pièce du puzzle en bas
   - Déplacer le slider
   - La pièce se déplace horizontalement
   - Cliquer "Vérifier"
