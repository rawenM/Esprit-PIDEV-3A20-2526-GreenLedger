📋 MÉTHODE RAPIDE: MySQL Workbench

1. Ouvrir MySQL Workbench
2. Se connecter
3. File → Open SQL Script
4. Sélectionner: generer_analyses_fraude_existantes.sql
5. Cliquer ⚡ Execute
6. ✅ Terminé!

📋 MÉTHODE ALTERNATIVE: phpMyAdmin

4. Copier-coller le SQL
