🎯 ERREUR

java.lang.ClassNotFoundException: javafx.embed.swing.SwingFXUtils

═══════════════════════════════════════════════════════════

🔍 CAUSE

Le module javafx.swing n'est pas disponible dans votre 
configuration JavaFX. SwingFXUtils est utilisé pour convertir 
BufferedImage (AWT) en Image (JavaFX).

═══════════════════════════════════════════════════════════

✅ SOLUTION APPLIQUÉE

Modification de PuzzleCaptchaService.base64ToImage():

AVANT:
```java
byte[] bytes = Base64.getDecoder().decode(base64);
ByteArrayInputStream bais = new ByteArrayInputStream(bytes);
BufferedImage bufferedImage = ImageIO.read(bais);
return SwingFXUtils.toFXImage(bufferedImage, null);
```

APRÈS:
```java
byte[] bytes = Base64.getDecoder().decode(base64);
ByteArrayInputStream bais = new ByteArrayInputStream(bytes);
return new Image(bais);
```

✅ Suppression de l'import javafx.embed.swing.SwingFXUtils
